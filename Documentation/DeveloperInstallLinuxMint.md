# Greg Finzer's .NET  Developer Install for Linux Mint



Enter this first into a bash shell:

```bash
bash
cd ~
sudo apt update 
sudo apt install git -y
mkdir aGit
cd aGit
git clone https://github.com/GregFinzer/Linux.git
cd Linux/Scripts
chmod +x SetupLinuxMintPart1.sh

```

Next enter into a bash shell:
```bash
./SetupLinuxMintPart1.sh
```
